# Research_XLSX

This project is for analyzing and processing data from the `Sukuk_NTS.xlsx` file using Python. The main script is `sukuk.py`.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the main script:
   ```bash
   python sukuk.py
   ```

## Files

- `sukuk.py`: Main Python script for data processing.
- `Sukuk_NTS.xlsx`: Excel data file.
